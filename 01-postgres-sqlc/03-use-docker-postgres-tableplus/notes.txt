Vamos agora, istalar o table plus, criar um container postgres, criar um esquema de banco de dados

'No Ubuntu '''''''''''''''''''''''''''''''''''''

Vindo no DockerHub: https://hub.docker.com/_/postgres e procurando a imagem do postgres
Instalamos a imagem usando o comando: docker pull postgres:14.20-trixie

Criando container postgres (ver a documentação no link acima)
docker run --name postgres14 -p 5432:5432 -e POSTGRES_USER=root -e POSTGRES_PASSWORD=secret -d postgres:14.20-trixie
(caso erre algo e precise apagar)
liste os containers: docker ps -a 
pare o container: docker stop (id container)
apague o container: docker rm (id container) 

Iniciando container: docker start postgres14
Executando container: docker exec -it postgres14 psql -U root
Sair: \q
